# Enterprise MIS & KPI Backend

## Tech Stack
- Java 17
- Spring Boot 3.x
- MySQL 8
- Flyway
- Spring Data JPA
- Spring Security
- Swagger

## Features
- Department master data
- Monthly targets & actuals
- KPI calculation engine
- Exception monitoring
- MIS reports

## How to Run
1. Create MySQL DB `mis_kpi_db`
2. Update credentials in `application.yml`
3. Run Spring Boot app
4. Access Swagger at `/swagger-ui.html`
